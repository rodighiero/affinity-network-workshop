Four tools, each running live in your browser: read one text under the microscope, map a set of texts or sounds with the telescope, or compare two documents side by side. Nothing is ever uploaded.

Text

Drop in one document and read it in layers: measurements, vocabulary, grammar, entities, readability, sentiment, keywords, recurring phrases, and a paragraph-by-paragraph narrative arc.

Network

Drop in a set of text files and watch them arrange into a similarity network. Embedding, similarity, and a live force layout all run in the browser; tune the threshold and forces, then export a PNG.

Audio

Drop in a set of short audio clips and watch them arrange by how they sound. The same network engine, embedding with CLAP instead of text — click a node to hear its clip.

Compare

Drop one file into each of two slots and every paragraph becomes a node, coloured by its source. Cross-document bridges mark shared themes; a panel below lists each document's own words and the words they share.
